# Kabini River Resort Website

A **static resort website** developed for **Kabini River Resort**, designed to showcase accommodations, amenities, gallery, and contact information with a clean, responsive, and user-friendly design.

🌐 **Live Website:** https://kabiniriverresort.com

---

## 📖 Project Overview

This project represents the official website of Kabini River Resort.  
It provides visitors with clear information about the resort, room types, activities, gallery, and booking-related pages.

The website is fully static and optimized for:
- Fast loading
- Simple navigation
- Cross-device compatibility

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  
- jQuery

## ✨ Features

- 📱 Fully responsive layout
- 🏨 Resort-focused UI/UX
- 🖼️ Image gallery

- ⚡ Fast-loading static pages
- 🌍 SEO-friendly structure
